From the picture we can see that the entropy $ H(P|C) = e + 0 $. The other entropy $H(K|C)$ is equal to $e+f$. The symbol stands for entropy $H(K|C,P)$ and as we know that entropy is always greater or equal to zero we can simply from the symbol state that $H(P|C) \leq H(K|C)$.

\item The second task is to answer the question when holds true that $H(K|C) = H(P|C)$. To full fill this equality it is necessary that the value for $f$ is also zero. This condition hold for such ciphers that when we know both cipher text and a plain text that we can say what the key is.  Such ciphers are for example the simple shift ciphers like Caesar cipher of affine cipher.
